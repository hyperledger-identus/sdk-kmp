createPrivateKey

open override fun createPrivateKey(properties: Map<String, Any>): PrivateKey(source)

Creates a private key based on the provided properties.

Return

The created private key.

Parameters

properties

A map of properties used to create the private key. The map should contain the following keys: - "type" (String): The type of the key ("EC" or "Curve25519"). - "curve" (String): The curve of the key. - "rawKey" (ByteArray): The raw key data (optional). - "index" (Int): The index for the key (only applicable for EC keys with curve "secp256k1"). - "derivationPath" (String): The derivation path for the key (only applicable for EC keys with curve "secp256k1"). - "seed" (String): The seed for the key (only applicable for EC keys with curve "secp256k1").

Throws

ApolloError.InvalidKeyType

If the provided key type is invalid.

ApolloError.InvalidKeyCurve

If the provided key curve is invalid.

ApolloError.InvalidRawData

If the provided raw key data is invalid.

ApolloError.InvalidIndex

If the provided index is invalid.

ApolloError.InvalidDerivationPath

If the provided derivation path is invalid.

ApolloError.InvalidSeed

If the provided seed is invalid.
